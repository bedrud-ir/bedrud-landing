لینک جلسه را در مرورگر باز کنید تا بپیوندید - نصب لازم نیست. برای تجربه اپلیکیشن بومی، کلاینت را برای پلتفرم خود در زیر نصب کنید.

---

## مرورگر (بدون نصب)

URL جلسه را در **Chrome**، **Firefox**، **Edge**، یا **Safari** باز کنید. دسترسی به دوربین و میکروفون نیاز به HTTPS دارد (ادمین سرور شما این را مدیریت می‌کند).

بدون دانلود، بدون افزونه، بدون حساب مورد نیاز اگر دسترسی مهمان فعال باشد.

## دسکتاپ Linux

<Tabs>
  <TabsList>
    <TabsTrigger value="appimage-recommended">AppImage (توصیه شده)</TabsTrigger>
    <TabsTrigger value="flatpak">Flatpak</TabsTrigger>
    <TabsTrigger value="snap">Snap</TabsTrigger>
    <TabsTrigger value="ubuntu-debian-apt">Ubuntu / Debian (apt)</TabsTrigger>
    <TabsTrigger value="arch-linux-aur">Arch Linux (AUR)</TabsTrigger>
    <TabsTrigger value="fedora-rhel-dnf">Fedora / RHEL (DNF)</TabsTrigger>
    <TabsTrigger value="portable-tarxz">Portable tar.xz</TabsTrigger>
  </TabsList>
  <TabsContent value="appimage-recommended">

    نصب لازم نیست. دانلود کنید، قابل اجرا کنید، اجرا کنید:

    ```bash
    wget https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.AppImage
    chmod +x bedrud-desktop-linux-x86_64.AppImage
    ./bedrud-desktop-linux-x86_64.AppImage
    ```

    <Callout type="tip">
        برای ادغام AppImage در راه‌انداز اپلیکیشن خود، از [AppImageLauncher](https://github.com/TheAssassin/AppImageLauncher) استفاده کنید.
    </Callout>
  </TabsContent>
  <TabsContent value="flatpak">

    ```bash
    flatpak install https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.flatpak
    flatpak run ir.bedrud.Desktop
    ```
  </TabsContent>
  <TabsContent value="snap">

    ```bash
    sudo snap install bedrud-desktop
    ```
  </TabsContent>
  <TabsContent value="ubuntu-debian-apt">

    اگر مخزن apt بدرود قبلاً روی سیستم شما پیکربندی شده است:

    ```bash
    sudo apt install bedrud-desktop
    ```

    برای تنظیم مخزن به [راهنمای نصب بسته](/fa/docs/guides/packages) مراجعه کنید.
  </TabsContent>
  <TabsContent value="arch-linux-aur">

    ```bash
    yay -S bedrud-desktop-bin
    ```
  </TabsContent>
  <TabsContent value="fedora-rhel-dnf">

    اگر مخزن DNF بدرود قبلاً پیکربندی شده است:

    ```bash
    sudo dnf install bedrud-desktop
    ```

    برای تنظیم مخزن به [راهنمای نصب بسته](/fa/docs/guides/packages) مراجعه کنید.
  </TabsContent>
  <TabsContent value="portable-tarxz">

    ```bash
    wget https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.tar.xz
    tar xf bedrud-desktop-linux-x86_64.tar.xz
    ./bedrud-desktop
    ```
  </TabsContent>
</Tabs>

## اتصال به سرور

وقتی برای اولین بار اپلیکیشن دسکتاپ را باز می‌کنید، از شما URL سرور درخواست می‌شود.

۱. URL ارائه شده توسط ادمین سرور خود را وارد کنید (مثلاً `https://meet.example.com`)
۲. ثبت‌نام کنید یا وارد شوید
۳. اتاق جلسه ایجاد کنید یا به آن بپیوندید

اپلیکیشن دسکتاپ اتصال به **چندین سرور** را پشتیبانی می‌کند. موارد بیشتر را در **تنظیمات** اضافه کنید.

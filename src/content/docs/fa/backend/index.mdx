## مقدمه

بک‌اند Bedrud یک پلتفرم جلسه تک‌باینری نوشته شده در **Go 1.24+** است. تمام وابستگی‌ها - سرور رسانه، فرانت‌اند وب، سرور TURN - در یک فایل اجرایی جاسازی شده‌اند.

### فناوری‌های کلیدی

- **فریمورک اصلی:** [Fiber v2](https://gofiber.io/) (روتر بدون تخصیص حافظه، API شبیه Express).
- **لایه پایگاه داده:** [GORM](https://gorm.io/) با پشتیبانی از **SQLite** (استاندارد) و **PostgreSQL** (تولید).
- **موتور رسانه:** [LiveKit](https://livekit.io/) (جاسازی‌شده به عنوان یک باینری خواهر).
- **سیستم احراز هویت:** احراز هویت چندلایه پشتیبانی‌کننده JWT، OAuth2 (Google/GitHub/Twitter)، و کلیدهای عبور FIDO2 بدون رمز عبور.
- **دارایی‌های استاتیک جاسازی‌شده:** از بسته `embed` Go 1.16+ برای بسته‌بندی فرانت‌اند React (دارایی‌های کلایت از پیش رندرشده SSR) استفاده می‌کند.
- **استقرار:** نصب‌کننده خودکار Debian/Ubuntu با ارکستراسیون systemd و پشتیبانی ACME (Let's Encrypt).

## چرا این معماری؟

پلتفرم‌های WebRTC معمولاً سرویس‌های جداگانه‌ای برای سیگنالینگ، TURN، سرویس وب و مدیریت پایگاه داده اجرا می‌کنند. Bedrud این‌ها را با روش‌های زیر در یک باینری ادغام می‌کند:

۱. **استخراج** باینری سرور رسانه در زمان اجرا.
۲. **پروکسی کردن** ترافیک رسانه از طریق پورت اصلی HTTP(S).
۳. **اتوماسیون** پیکربندی سطح سیستم (SSL، systemd) از طریق خود باینری.

## معماری سطح بالا

بک‌اند یک معماری لایه‌ای استاندارد دنبال می‌کند:

۱. **لایه سرور (`internal/server`):** برنامه Fiber، مسیرها و میان‌افزار را تنظیم می‌کند.
۲. **لایه هندلر (`internal/handlers`):** درخواست‌ها و پاسخ‌های HTTP را مدیریت می‌کند.
۳. **لایه مخزن (`internal/repository`):** پرس‌وجوهای پایگاه داده را با استفاده از GORM مدیریت می‌کند.
۴. **لایه مدل (`internal/models`):** جداول پایگاه داده و ساختارهای Go را تعریف می‌کند.
۵. **لایه سرویس/منطق:** برای سادگی در هندلرها و مخازن ادغام شده است، با بسته‌های تخصصی مانند `internal/auth` و `internal/livekit`.

## نقطه ورودی اصلی

برنامه در `server/cmd/bedrud/main.go` شروع می‌شود. سه حالت اصلی دارد:

- `run`: سرور کامل جلسات را شروع می‌کند.
- `livekit`: فقط سرور رسانه جاسازی‌شده را شروع می‌کند.
- `install`: سرور را به صورت خودکار در سیستم‌های Debian/Ubuntu تنظیم می‌کند.

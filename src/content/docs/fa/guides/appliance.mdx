Bedrud برای اجرا به عنوان یک "لوازم" مستقل برای جلسات ویدیویی طراحی شده است. یک فایل اجرایی باینری همه چیز مورد نیاز را بسته‌بندی می‌کند - فرانت‌اند، بک‌اند و سرور رسانه LiveKit.

## ویژگی‌های کلیدی

| ویژگی | توضیحات |
|---------|-------------|
| بدون وابستگی خارجی | نیازی به Node.js، Redis یا سرور رسانه جداگانه نیست |
| سرور رسانه جاسازی‌شده | باینری LiveKit شامل و به صورت خودکار مدیریت می‌شود |
| فرانت‌اند جاسازی‌شده | UI React کامپایل و SSR از پیش رندرشده در باینری Go |
| ذخیره‌سازی SQLite | نیازی به سرور پایگاه داده نیست |
| TLS داخلی | گواهی‌های امضاشده خود یا Let's Encrypt |
| نصب‌کننده داخلی | پیکربندی systemd، دایرکتوری‌ها و تنظیمات |

## اجرای باینری

### شروع سرور Bedrud

```bash
./bedrud --run --config config.yaml
```

### شروع سرور رسانه LiveKit

```bash
./bedrud --livekit --config livekit.yaml
```

باینری شامل هر دو سرور API و سرور رسانه است. از پرچم‌ها برای انتخاب کدام‌یک را شروع کنید استفاده کنید.

## نصب

### نصب سریع (Debian/Ubuntu)

```bash
# با TLS Let's Encrypt
sudo ./bedrud install --tls --domain meet.example.com --email admin@example.com

# با گواهی امضاشده خود
sudo ./bedrud install --tls --ip 1.2.3.4

# HTTP ساده (فقط توسعه)
sudo ./bedrud install --ip 1.2.3.4
```

### معماری سرویس

پس از نصب، دو سرویس systemd اجرا می‌شوند:

```
┌──────────────────┐     ┌──────────────────┐
│  bedrud.service  │     │  livekit.service  │
│                  │     │                   │
│  bedrud --run    │────►│  bedrud --livekit │
│  (API + Web UI)  │     │  (WebRTC media)   │
└──────────────────┘     └───────────────────┘
          │
          ▼
    ┌──────────┐
    │  SQLite  │
    │ database │
    └──────────┘
```

## فایل‌های پیکربندی

| فایل | هدف |
|------|---------|
| `/etc/bedrud/config.yaml` | پیکربندی اصلی سرور |
| `/etc/bedrud/livekit.yaml` | پیکربندی سرور رسانه |
| `/var/lib/bedrud/bedrud.db` | پایگاه داده SQLite |
| `/var/log/bedrud/bedrud.log` | لاگ‌های برنامه |

به [مرجع پیکربندی](/docs/getting-started/configuration) برای همه گزینه‌ها مراجعه کنید.

### بررسی وضعیت سرویس

```bash
systemctl status bedrud livekit
```

```bash
tail -f /var/log/bedrud/bedrud.log
journalctl -u bedrud -f
```

## حذف نصب

```bash
sudo ./bedrud uninstall
```

این به طور کامل حذف می‌کند:

- فایل‌های سرویس systemd
- باینری از `/usr/local/bin/`
- پیکربندی در `/etc/bedrud/`
- داده‌ها در `/var/lib/bedrud/`
- لاگ‌ها در `/var/log/bedrud`

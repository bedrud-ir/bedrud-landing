## Seguridad y TLS

El instalador maneja la seguridad automáticamente:

- **ACME (Let's Encrypt):** Si proporcionas un dominio y correo electrónico, configura la renovación automática de certificados.
- **Certificados Autofirmados:** Si no se proporciona un dominio, genera un certificado autofirmado para que el sitio aún funcione a través de HTTPS.
- **Proxy Inverso:** El backend actúa como un proxy inverso. Recibe tráfico en el puerto 443 y envía solicitudes `/livekit` al servidor de medios interno.

## Desinstalación

Si necesitas eliminar Bedrud, puedes ejecutar:
```bash
sudo bedrud uninstall
```
Esto detiene los servicios, elimina los archivos de systemd y elimina las carpetas de configuración y datos.

Ouvrez le lien de réunion dans un navigateur pour rejoindre - aucune installation requise. Pour une expérience d'application native, installez le client pour votre plateforme ci-dessous.

---

## Navigateur (aucune installation)

Ouvrez l'URL de réunion dans **Chrome**, **Firefox**, **Edge** ou **Safari**. L'accès à la caméra et au microphone nécessite HTTPS (votre administrateur de serveur gère cela).

Aucun téléchargement, aucune extension, aucun compte requis si l'accès invité est activé.

## Linux Desktop

<Tabs>
  <TabsList>
    <TabsTrigger value="appimage-recommended">AppImage (recommandé)</TabsTrigger>
    <TabsTrigger value="flatpak">Flatpak</TabsTrigger>
    <TabsTrigger value="snap">Snap</TabsTrigger>
    <TabsTrigger value="ubuntu-debian-apt">Ubuntu / Debian (apt)</TabsTrigger>
    <TabsTrigger value="arch-linux-aur">Arch Linux (AUR)</TabsTrigger>
    <TabsTrigger value="fedora-rhel-dnf">Fedora / RHEL (DNF)</TabsTrigger>
    <TabsTrigger value="portable-tarxz">Portable tar.xz</TabsTrigger>
  </TabsList>
  <TabsContent value="appimage-recommended">

    Aucune installation requise. Téléchargez, rendez exécutable, exécutez :

    ```bash
    wget https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.AppImage
    chmod +x bedrud-desktop-linux-x86_64.AppImage
    ./bedrud-desktop-linux-x86_64.AppImage
    ```

    <Callout type="tip">
        Pour intégrer AppImage dans votre lanceur d'application, utilisez [AppImageLauncher](https://github.com/TheAssassin/AppImageLauncher).
    </Callout>
  </TabsContent>
  <TabsContent value="flatpak">

    ```bash
    flatpak install https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.flatpak
    flatpak run ir.bedrud.Desktop
    ```
  </TabsContent>
  <TabsContent value="snap">

    ```bash
    sudo snap install bedrud-desktop
    ```
  </TabsContent>
  <TabsContent value="ubuntu-debian-apt">

    Si le dépôt apt Bedrud est déjà configuré sur votre système :

    ```bash
    sudo apt install bedrud-desktop
    ```

    Voir le [guide d'installation des packages](/fr/docs/guides/packages) pour la configuration du dépôt.
  </TabsContent>
  <TabsContent value="arch-linux-aur">

    ```bash
    yay -S bedrud-desktop-bin
    ```
  </TabsContent>
  <TabsContent value="fedora-rhel-dnf">

    Si le dépôt DNF Bedrud est déjà configuré :

    ```bash
    sudo dnf install bedrud-desktop
    ```

    Voir le [guide d'installation des packages](/fr/docs/guides/packages) pour la configuration du dépôt.
  </TabsContent>
  <TabsContent value="portable-tarxz">

    ```bash
    wget https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.tar.xz
    tar xf bedrud-desktop-linux-x86_64.tar.xz
    ./bedrud-desktop
    ```
  </TabsContent>
</Tabs>

    ```bash
    curl -L https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-macos-arm64.tar.gz | tar xz
    xattr -d com.apple.quarantine bedrud-desktop
    ./bedrud-desktop
    ```
  </TabsContent>
  <TabsContent value="intel-x8664">

    ```bash
    curl -L https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-macos-x86_64.tar.gz | tar xz
    xattr -d com.apple.quarantine bedrud-desktop
    ./bedrud-desktop
    ```

</Tabs>

<Callout type="warning">
    Les builds macOS peuvent être **non signés**. Si Gatekeeper bloque l'application, exécutez la commande `xattr` ci-dessus, ou allez dans **Réglages système → Confidentialité et sécurité** et cliquez sur **Ouvrir quand même**.
</Callout>

## Connexion à un serveur

Lorsque vous ouvrez l'application de bureau pour la première fois, vous serez invité à entrer une URL de serveur.

1. Entrez l'URL fournie par votre administrateur de serveur (ex. `https://meet.example.com`)
2. Enregistrez un compte ou connectez-vous
3. Créez ou rejoignez une salle de réunion

L'application de bureau prend en charge la connexion à **plusieurs serveurs**. Ajoutez-en d'autres dans **Paramètres**.

Katılmak için toplantı bağlantısını bir tarayıcıda açın - kurulum gerekmez. Yerel bir uygulama deneyimi için aşağıdan platformunuza uygun istemciyi kurun.

---

## Tarayıcı (Sıfır Kurulum)

Toplantı URL'sini **Chrome**, **Firefox**, **Edge** veya **Safari**'de açın. Kamera ve mikrofon erişimi HTTPS gerektirir (sunucu yöneticiniz bunu halleder).

İndirme gerekmez, eklenti gerekmez, konuk erişimi etkinse hesap gerekmez.

## Linux Masaüstü

<Tabs>
  <TabsList>
    <TabsTrigger value="appimage-recommended">AppImage (önerilen)</TabsTrigger>
    <TabsTrigger value="flatpak">Flatpak</TabsTrigger>
    <TabsTrigger value="snap">Snap</TabsTrigger>
    <TabsTrigger value="ubuntu-debian-apt">Ubuntu / Debian (apt)</TabsTrigger>
    <TabsTrigger value="arch-linux-aur">Arch Linux (AUR)</TabsTrigger>
    <TabsTrigger value="fedora-rhel-dnf">Fedora / RHEL (DNF)</TabsTrigger>
    <TabsTrigger value="portable-tarxz">Taşınabilir tar.xz</TabsTrigger>
  </TabsList>
  <TabsContent value="appimage-recommended">

    Kurulum gerekmez. İndirin, çalıştırılabilir yapın, çalıştırın:

    ```bash
    wget https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.AppImage
    chmod +x bedrud-desktop-linux-x86_64.AppImage
    ./bedrud-desktop-linux-x86_64.AppImage
    ```

    <Callout type="tip">
        AppImage'ı uygulama başlatıcınıza entegre etmek için [AppImageLauncher](https://github.com/TheAssassin/AppImageLauncher) kullanın.
    </Callout>
  </TabsContent>
  <TabsContent value="flatpak">

    ```bash
    flatpak install https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.flatpak
    flatpak run ir.bedrud.Desktop
    ```
  </TabsContent>
  <TabsContent value="snap">

    ```bash
    sudo snap install bedrud-desktop
    ```
  </TabsContent>
  <TabsContent value="ubuntu-debian-apt">

    Bedrud apt deposu sisteminizde zaten yapılandırılmışsa:

    ```bash
    sudo apt install bedrud-desktop
    ```

    Depo kurulumu için bkz. [Paket Kurulumu kılavuzu](/tr/docs/guides/packages).
  </TabsContent>
  <TabsContent value="arch-linux-aur">

    ```bash
    yay -S bedrud-desktop-bin
    ```
  </TabsContent>
  <TabsContent value="fedora-rhel-dnf">

    Bedrud DNF deposu zaten yapılandırılmışsa:

    ```bash
    sudo dnf install bedrud-desktop
    ```

    Depo kurulumu için bkz. [Paket Kurulumu kılavuzu](/tr/docs/guides/packages).
  </TabsContent>
  <TabsContent value="portable-tarxz">

    ```bash
    wget https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-linux-x86_64.tar.xz
    tar xf bedrud-desktop-linux-x86_64.tar.xz
    ./bedrud-desktop
    ```
  </TabsContent>
</Tabs>

## macOS

<Tabs>
  <TabsList>
    <TabsTrigger value="homebrew-recommended">Homebrew (önerilen)</TabsTrigger>
    <TabsTrigger value="dmg-apple-silicon">Apple Silicon (.dmg)</TabsTrigger>
    <TabsTrigger value="dmg-intel">Intel (.dmg)</TabsTrigger>
    <TabsTrigger value="apple-silicon-arm64">Apple Silicon (tar.gz)</TabsTrigger>
    <TabsTrigger value="intel-x8664">Intel (tar.gz)</TabsTrigger>
  </TabsList>
  <TabsContent value="homebrew-recommended">

    ```bash
    brew tap bedrud-ir/bedrud
    brew install bedrud-desktop
    ```
  </TabsContent>
  <TabsContent value="dmg-apple-silicon">

    DMG'yi [GitHub Sürümleri](https://github.com/bedrud-ir/bedrud/releases/latest) sayfasından indirin:

    | Mimari | Dosya |
    |---|---|
    | Apple Silicon | `bedrud-desktop-macos-arm64.dmg` |

    DMG'yi açın, Bedrud'u Uygulamalar'a sürükleyin. Gatekeeper uygulamayı engellerse, **Sistem Ayarları → Gizlilik ve Güvenlik**'e gidin ve **Yine de Aç**'a tıklayın.
  </TabsContent>
  <TabsContent value="dmg-intel">

    DMG'yi [GitHub Sürümleri](https://github.com/bedrud-ir/bedrud/releases/latest) sayfasından indirin:

    | Mimari | Dosya |
    |---|---|
    | Intel | `bedrud-desktop-macos-x86_64.dmg` |

    DMG'yi açın, Bedrud'u Uygulamalar'a sürükleyin. Gatekeeper uygulamayı engellerse, **Sistem Ayarları → Gizlilik ve Güvenlik**'e gidin ve **Yine de Aç**'a tıklayın.
  </TabsContent>
  <TabsContent value="apple-silicon-arm64">

    ```bash
    curl -L https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-macos-arm64.tar.gz | tar xz
    xattr -d com.apple.quarantine bedrud-desktop
    ./bedrud-desktop
    ```
  </TabsContent>
  <TabsContent value="intel-x8664">

    ```bash
    curl -L https://github.com/bedrud-ir/bedrud/releases/latest/download/bedrud-desktop-macos-x86_64.tar.gz | tar xz
    xattr -d com.apple.quarantine bedrud-desktop
    ./bedrud-desktop
    ```
  </TabsContent>
</Tabs>

<Callout type="warning">
    macOS derlemeleri **imzalanmamış** olabilir. Gatekeeper uygulamayı engellerse, yukarıdaki `xattr` komutunu çalıştırın veya **Sistem Ayarları → Gizlilik ve Güvenlik**'e gidip **Yine de Aç**'a tıklayın.
</Callout>

## Bir Sunucuya Bağlanma

Masaüstü uygulamasını ilk açtığınızda bir sunucu URL'si istenir.

1. Sunucu yöneticiniz tarafından sağlanan URL'yi girin (örn. `https://meet.example.com`)
2. Bir hesap kaydedin veya giriş yapın
3. Bir toplantı odası oluşturun veya katılın

Masaüstü uygulaması **birden fazla sunucuya** bağlanmayı destekler. Daha fazlasını **Ayarlar**'dan ekleyin.

Bedrud, konteynerlerde derlemek ve çalıştırmak için çok aşamalı bir Dockerfile sağlar.

## Hızlı Başlangıç

```bash
# Derle
docker build -t bedrud .

# Çalıştır
docker run -d --name bedrud \
  -p 8090:8090 \
  -p 7880:7880 \
  -p 50000-60000:50000-60000/udp \
  bedrud
```

## Dockerfile Genel Bakış

Dockerfile, son imaj boyutunu küçültmek için üç aşama kullanır:

### Aşama 1: Ön Uç Derlemesi

```dockerfile
FROM node:22-alpine AS frontend
RUN npm install -g bun
WORKDIR /build/apps/web
COPY apps/web/package.json apps/web/bun.lock ./
RUN bun install --frozen-lockfile
COPY apps/web/ ./
RUN bun run build:embed
```

Bun'u kurar, React uygulamasını SSR ile ön rendeleyip varlıkları `server/frontend/` dizinine kopyalar.

### Aşama 2: Sunucu Derlemesi

```dockerfile
FROM golang:1.24-alpine AS backend
RUN apk add --no-cache gcc musl-dev
WORKDIR /build/server
COPY server/go.mod server/go.sum ./
RUN go mod download
COPY server/ ./
COPY --from=frontend /build/server/frontend ./frontend/
RUN CGO_ENABLED=1 GOOS=linux go build -ldflags="-s -w" -o /bedrud ./cmd/bedrud/main.go
# CGO is required for SQLite (uses C bindings)
```

Go ikili dosyasını gömülü ön uçla birlikte derler. SQLite desteği için CGO etkinleştirilmiştir.

### Aşama 3: Çalışma Zamanı

```dockerfile
FROM alpine:3.21
RUN apk add --no-cache ca-certificates tzdata
COPY --from=backend /bedrud /usr/local/bin/bedrud
EXPOSE 8090 7880
ENTRYPOINT ["bedrud"]
CMD ["run"]
```

Yalnızca ikili dosya, CA sertifikaları ve saat dilimi verilerini içeren minimal Alpine imajı.

## Portlar

| Port          | Servis        | Protokol          |
|---------------|---------------|-------------------|
| 8090          | API + Web UI  | HTTP              |
| 7880          | LiveKit       | WebSocket + HTTP  |
| 50000-60000   | RTC medya     | UDP               |

LiveKit medya akışları için RTC UDP port aralığını da (varsayılan: 50000-60000) açmanız gerekebilir.

## Birimler

Kalıcı veriler için bir birim bağlayın:

```bash
docker run -d \
  --name bedrud \
  -p 8090:8090 \
  -p 7880:7880 \
  -v bedrud-data:/var/lib/bedrud \
  bedrud
```

## Yapılandırma

Özel bir yapılandırma dosyası belirtin:

```bash
docker run -d \
  --name bedrud \
  -p 8090:8090 \
  -p 7880:7880 \
  -v /path/to/config.yaml:/etc/bedrud/config.yaml \
  -v bedrud-data:/var/lib/bedrud \
  bedrud run --config /etc/bedrud/config.yaml
```

Veya ortam değişkenlerini kullanın:

```bash
docker run -d \
  --name bedrud \
  -p 8090:8090 \
  -p 7880:7880 \
  -e JWT_SECRET=my-production-secret \
  -e LIVEKIT_API_KEY=prodkey \
  -e LIVEKIT_API_SECRET=prodsecret \
  bedrud
```

## Önceden Oluşturulmuş İmajlar

Docker imajları her sürümde GitHub Container Registry'e yayınlanır:

```bash
# En son sürüm
docker pull ghcr.io/bedrud-ir/bedrud:latest

# Belirli bir sürüm
docker pull ghcr.io/bedrud-ir/bedrud:v1.0.0
```

## Docker Compose Örneği

```yaml
services:
  bedrud:
    image: ghcr.io/bedrud-ir/bedrud:latest
    ports:
      - "8090:8090"
      - "7880:7880"
    volumes:
      - bedrud-data:/var/lib/bedrud
      - ./config.yaml:/etc/bedrud/config.yaml
    environment:
      - JWT_SECRET=change-me
    restart: unless-stopped

volumes:
  bedrud-data:
```

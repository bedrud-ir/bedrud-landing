# Bedrud Agent Instructions

## Build & Dev

**Required pre-step:** Search index auto-generates on `bun dev` / `bun run build` via `bun scripts/generate-search-index.ts`. Outputs per-locale JSON to `public/`.

**CI order:** `bun run check` → `bun run typecheck` → `bun test` → `bun run build`. Follow locally.

**Single test:** `bun test path/to/test.test.ts`

**Type check:**
- `bun run typecheck` - TS compiler
- `bun run typecheck:astro` - Astro checker

## i18n

**Build-time only.** No runtime lib. Use `t(locale, "key.subkey")` from `src/i18n/utils.ts`.

**9 locales:** en, de, fr, es, zh, ja, tr, fa, ar. Routes under `/:lang/`. `/` → `/en`.

**RTL:** fa, ar auto-set `dir="rtl"` via `getDir(locale)`.

**Translations:** `src/i18n/locales/{lang}.ts` — nested objects, dot-notation.

## Architecture

**Path alias:** `~/` → `src/`

**Content:** MDX docs in `src/content/docs/`. Schema: `src/content.config.ts`.

**Sidebar:** `src/content/docs/sidebar.ts`. Generator (`scripts/sync-sidebar.ts`) broken — references missing `sections.ts`. **Do NOT fix or run.**

**Icons:** lucide-react only.

**Styling:** Tailwind v4 + CSS vars. Global: `src/styles/global.css`. ScrollArea = CSS-only (`.scroll-area`), not Radix.

## Code Conventions

**Components:** shadcn/ui (CVA + Tailwind). Merge classes with `cn()` from `src/lib/utils.ts`.

**Animations:**
- Scroll-linked (macbook-scroll, mobile-phone-scroll): React islands in `src/components/ui/`
- All others: CSS keyframes + `data-animate` / `.in-view` (IntersectionObserver in Base.astro)

**Biome:** Format + lint. Config: `biome.json`. Double quotes, semicolons, 2-space indent.
- `bun run check` — lint + format check
- `bun run format` — auto-format
- `bun run lint` — lint only

**Testing:** `bun:test` + `@happy-dom/global-registrator` (setup: `test/setup.ts`). Mirror `src/` in `test/`.

## Generated Files

**Search indexes:** `public/search-index-{locale}.json` — auto-generated by `scripts/generate-search-index.ts`. **Do NOT edit.**

**Biome excludes:** `public/search-index-*.json`, `.astro/`, `dist/`, `node_modules/`.

## Scripts

- `scripts/generate-search-index.ts` — run before dev/build
- `scripts/sync-sidebar.ts` — **BROKEN** (missing `sections.ts`). Do NOT use.
- `scripts/audit-frontmatter.ts` — validates MDX frontmatter (title, description)

## Commit Message Guide

**Format:**

```
<action> <what> for <why>
```

**Actions:** `add` (new), `delete` (removed), `update` (changed)

**Examples:**

```
add commit template for consistent git history
delete unused imports for cleaner module output
update README for new install instructions
add user model for auth feature
delete legacy helpers for migration to Go
update error handling for clearer debug logs
```
